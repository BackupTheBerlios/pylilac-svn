Package pylilac



The PYthon LInguistic Laboratory (pylilac) is a Python library and a set of tools to help in studying, analyzing and engineering languages. In the initial study of a natural or constructed language, the need for a framework to store the language characteristics may rise. In particular when studying several languages, it can be useful a tool to bring the study to a normalized form, to highlight the things in common with other languages. PyLilac wants to provide a library and a set of tools to study, analyze and engineer languages as composed by:
  • A lexicon
  • An inflective system
  • A grammar
When exposed to a new language, the user will be able to describe it and, later on, he will be able to browse through the schema they have built. On the other hand, another user will be able to open the same file and to explore the language prepared by someone else. They both will be going to use a graphical interface tool to work with the data. Eventually, a third user might want to develop a linguistic software (for training purpose, documentation, study and so on) and he will be going to use the library directly.

Submodules
  • pylilac.core: Core library module for pylilac
    • pylilac.core.bnf: A module to build expressions in the Extended Backus-Naur Form (EBNF).
    • pylilac.core.expression: A module for expression management.
    • pylilac.core.fsa: A module for grammar compiling and for parsing.
    • pylilac.core.grammar: A module to define and manage grammar rules.
    • pylilac.core.inflection: A module for the generation of inflected forms.
    • pylilac.core.interlingua: A module to manage the machine interlingua used to express universal concepts.
    • pylilac.core.lect: A module for serialization and management of a language variety.
    • pylilac.core.lexicon: A module for lexicon management: Word and Lemma.
    • pylilac.core.optiontree: A module for the OptionTree utility class.
    • pylilac.core.tokenizer: A specialized parser for expressions.
    • pylilac.core.utilities: Common utilities functions.
  • pylilac.ui: Module for the graphic user interface.
    • pylilac.ui.graphics: A module to wrap the images used in wx graphical interface.
    • pylilac.ui.la: A module for the wx frames in the Language Architect.
    • pylilac.ui.lacodebehind: Code behind for the Language Laboratory.
    • pylilac.ui.lawidgets: A module for the wx widgets used in the Language Architect.
